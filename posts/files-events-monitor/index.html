Are you concerned about the integrity and security of your files? Do you want to keep track of any changes or activities in a specific directory? Introducing Files Events Monitor, a simple yet powerful tool to monitor file system activity and receive real-time alerts via Discord webhook.

Files Events Monitor is a Python-based command-line tool designed to keep a vigilant eye on your specified directory for any file system events. It provide you real-time notifications and logging of crucial events, enabling you to respond promptly to any changes or suspicious activities.

Features#

  • Real-time monitoring of file system events.
  • Integration with Discord webhooks for immediate alerts.
  • Option to save event logs to a file for later review.
